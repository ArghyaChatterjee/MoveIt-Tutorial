# MoveIt2 with Isaac Sim

*Easy-to-use open source robotics manipulation platform for developing commercial applications, prototyping designs, and benchmarking algorithms.*


## Getting Started

See our extensive [Tutorials and Documentation](https://moveit.picknik.ai/).


## Requirements

This repository is tested on ubuntu 22.04, cuda 12.1, Python 3.10, Isaac Sim 4.5 and ROS2 Humble. 

## Download
First download Omniverse Launcher from this [site](https://developer.nvidia.com/omniverse?sortBy=developer_learning_library%2Fsort%2Ffeatured_in.omniverse%3Adesc%2Ctitle%3Aasc#section-getting-started) for ubuntu / linux distribution.

## Install
To start the installation for Nvidia Omniverse Isaac Sim, run:
```bash
sudo apt install libfuse2
```
Permissions of the downloaded AppImage can be changed to run as an administrator in the following way:

```bash
chmod +x omniverse-launcher-linux.AppImage
```
Once those items are completed, you should be able to double-click on the AppImage package and have it run. Or, run it from the Terminal:

```bash
./omniverse-launcher-linux.AppImage
```

Install URDF related packages for internal conversions:
```bash
sudo apt install ros-humble-xacro
```
Now, install binary packages for MoveIt.
```bash
sudo apt install ros-humble-ros2-control ros-humble-ros2-controllers ros-humble-gripper-controllers ros-humble-moveit-py
```
